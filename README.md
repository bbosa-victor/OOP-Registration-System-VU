# Student Registration Form- Question 2
**GroupB Project | Victoria University**

## Description
This is a Java Swing application for registering new students. It features real-time validation, automatic ID generation (YYYY-xxxxx), and saves data to both a CSV file and a Microsoft Access database.

## Features Met
- [cite_start]**Auto-updating DOB**: The day count adjusts for Leap Years and Months[cite: 74].
- [cite_start]**Validation**: Enforces 8-20 character passwords and 16-60 age range[cite: 78, 79].
- [cite_start]**Data Storage**: Appends records to `students.csv` and updates `students_dbl`[cite: 81, 85].

## How to Run
1. Ensure the `UCanAccess` library is added to the project.
2. Place the `students_db.accdb` file in a folder named `database` at the root of the project.
3. Run the `StudentRegistrationForm2.java` file.
